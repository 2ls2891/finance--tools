개인정보처리방침

1. 개인정보의 수집 및 이용

본 웹사이트는 별도의 회원가입이나 로그인 절차가 없으며, 개인정보를 수집하거나 저장하지 않습니다.

2. 쿠키 및 로컬 스토리지

본 웹사이트는 다음과 같은 용도로 브라우저의 로컬 스토리지를 사용합니다:

  • 환율 정보 캐싱: 환율 API 호출을 최소화하고 빠른 로딩을 위해 30분간 환율 데이터를 저장합니다.
  • 저장된 데이터는 사용자의 기기에만 보관되며, 외부로 전송되지 않습니다.
  • 사용자는 브라우저 설정에서 언제든지 로컬 스토리지 데이터를 삭제할 수 있습니다.

3. 계산 데이터

사용자가 입력한 모든 계산 데이터(대출 금액, 적금액, 투자금 등)는:

  • 서버로 전송되지 않습니다
  • 브라우저 내에서만 처리됩니다
  • 페이지를 새로고침하면 모두 삭제됩니다
  • 제3자와 공유되지 않습니다

4. 외부 서비스

본 웹사이트는 다음의 외부 서비스를 사용합니다:

  • exchangerate-api.com: 실시간 환율 정보를 제공받습니다. 사용자의 개인정보는 전송되지 않습니다.
  • Google AdSense (향후 적용 예정): 광고 서비스를 위해 쿠키를 사용할 수 있습니다. 자세한 내용은 Google 개인정보처리방침을 참조하세요.

5. 분석 도구 (향후 적용 예정)

웹사이트 개선을 위해 Google Analytics와 같은 분석 도구를 사용할 수 있습니다. 이 경우:

  • 익명화된 방문 통계만 수집됩니다
  • IP 주소는 익명화되어 처리됩니다
  • 개인을 식별할 수 있는 정보는 수집되지 않습니다

6. 아동의 개인정보 보호

본 웹사이트는 만 14세 미만 아동의 개인정보를 수집하지 않으며, 별도의 회원가입이나 개인정보 입력이 필요하지 않습니다.

7. 개인정보처리방침 변경

본 방침은 법령이나 서비스 변경사항에 따라 수정될 수 있습니다. 중요한 변경사항이 있을 경우 웹사이트를 통해 공지하겠습니다.

최종 수정일: 2025년 1월 7일

8. 면책사항

본 웹사이트에서 제공하는 모든 계산 결과는 참고용이며, 실제 금융 상품 및 서비스와 차이가 있을 수 있습니다. 정확한 정보는 해당 금융기관에 문의하시기 바랍니다.

9. 문의

개인정보처리방침과 관련하여 궁금한 사항이 있으시면 GitHub Issues를 통해 문의해 주시기 바랍니다.
